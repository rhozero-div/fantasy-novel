# 写作规则

> 这里保留最重要的约束；具体语感与节奏优先参考 `writing-style-sample.md`。

## 故事分类
机甲 — 巨型机器人/宇宙战争/军事科幻/政治冲突

## 命名风格
西式

## 风格偏好
- 军事科幻语感优先于热血口号
- 战斗描写强调推进器、装甲、惯性、告警声等可感知物理细节
- 文戏避免空泛抒情，人物立场变化尽量通过沉默、迟疑和行动体现

## 禁区
- 不出现修仙/魔法/超自然元素
- 不出现现实国家名称
- 避免过度浪漫化的战争描写

## 参考范文

当前 demo 已提供 `skill_context/writing-style-sample.md`，写作时优先吸收其冷静、压迫、偏驾驶舱体感的表达方式。
