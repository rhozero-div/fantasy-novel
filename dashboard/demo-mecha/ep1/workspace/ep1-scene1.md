# Scene 1: 实验部队报到

天炉基地的整备库比艾雷想象中更安静。高处吊臂悬停在半空，像一群被人按住喉咙的钢铁鸟类，只剩下液压系统偶尔吐出一声低喘。艾雷站在玻璃隔墙外，第一次看见 XGF-01A 的全貌。灰白色装甲尚未完全闭合，背部推进器的保护盖还没有拆下，那台机体像一头沉睡中的巨兽，明明一动不动，却已经把整间整备库的空气压得发沉。

技术官把资料板递给他时，语气平得像在念一段保养手册。“实验机，调校未完，神经反馈阈值不稳定。”他说到这里停了一下，才补上一句：“不过你既然被分到这里，说明上面觉得你扛得住。”

这话听着像夸奖，落在耳里却更像提醒。艾雷把视线从机体上收回来，指尖在资料板边缘轻轻扣了一下，还是说了句“明白”。他知道自己是军校里最晚被定岗的那一批，也知道很多人把实验部队当作去处不明的收容站。可真正站到这台机体面前时，那点被挑剩下的不甘，还是被更直接的兴奋压了过去。

队长从后面走来，抬手在他肩上拍了一下，力道不重。“第一次上手别急着证明自己。”他笑了笑，语气比整备库里其他人都暖一点，“活着回来，比漂亮成绩更重要。”

艾雷点头时，目光还是忍不住又落回 XGF-01A 身上。那层尚未封闭的装甲缝隙里透着冷白灯光，像机体内部有什么东西也在打量他。他分不清那是期待，还是不安。
