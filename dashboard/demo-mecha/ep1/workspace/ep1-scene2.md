# Scene 2: 首次出击 — 巡逻

残骸带外缘的宇宙并不安静。推进器喷焰一亮起来，漂浮的金属碎片就会在黑暗里翻出一层微冷的光。艾雷跟在队长机后方，手心全是汗，却又止不住胸腔里越跳越快的兴奋。驾驶舱警示灯一枚一枚亮着，XGF-01A 的操纵反馈比训练机更敏锐，像他每一次呼吸都会被机体内部那些还没驯服的回路听见。

警报就是在这个时候骤然拉响的。

第一道光束从残骸缝隙里切出来时，艾雷几乎以为那只是雷达误报。可下一秒，队长机侧装甲被打得火花四溅，六个红点同时在屏幕边缘亮起，像一圈骤然收紧的兽眼。通讯频道里有人吼他规避，他本能地拉高机体，几乎是擦着一截断裂舰身翻过去。光束擦过盾牌边缘，整台机体都被震得发麻。

“用残骸做掩体，别停在空域中央！”队长的命令刚落下，第二轮齐射已经追了上来。艾雷把 XGF-01A 压进碎舰残骸后侧，推进器短促点火，在视野里拖出一串急促亮纹。他看见敌机在远处绕出新的角度，像一群早就踩熟地形的猎手。

就在一束主炮正面压来的瞬间，艾雷忽然感觉操纵杆下的阻力变了。不是他动作更快，而是机体先一步把那道死线让开。XGF-01A 以一种超出训练记录的速度侧闪，光束从肩甲外缘擦过去，驾驶舱里却只留下短促而锐利的蜂鸣。

艾雷怔了一瞬，随即听见自己更重的呼吸声。他没空细想机体为什么突然“活”过来，因为下一秒，队长机已经在正前方被贯穿，爆光把整片残骸带都照成了苍白色。通讯频道炸成一片，艾雷的喉咙却像被什么堵住了。他第一次真正明白，实战从来不是训练场上那些可以重来的科目。
