# Scene 3: 被迫脱离

队长机爆散后的残光还没完全熄灭，敌机已经把包围圈压得更小。XGF-01A 的光束步枪在最后一次点射后跳出空仓告警，驾驶舱里只剩艾雷自己的呼吸和关节马达越来越急的震鸣。警示界面不断闪烁，装甲受损、推进剂下降、外部温度骤降，每一条都像在逼他承认自己已经走到绝境。

艾雷看着屏幕上的三个红点逼近，忽然意识到再往后退，只会被一点点切碎在残骸堆里。他把空掉的步枪甩开，手指压上短刀启动键。刀刃弹出的瞬间，驾驶舱里亮起一道冷白线，短得像一口被人咬住的呼吸。

“他们没有我用命去开的决心。”这句话不是说给谁听，更像是他终于把自己逼到了必须相信的那一侧。

XGF-01A 猛地推满推进器，从残骸阴影里直冲出去。第一台敌机显然没料到他会放弃远射，机身刚要后撤，短刀已经从接缝处捅进装甲，金属被硬生生撬开的阻力顺着操纵杆一路震回手臂。艾雷几乎咬碎牙关才把刀抽出来，第二台敌机的枪口已经抬起，他借着翻滚姿态把一截断裂装甲板撞过去，硬生生逼得对方失去瞄准。

近战把一切都拉到了最粗暴的距离里。没有谁还能躲在雷达和准星后面，只剩下钢铁相撞时那种会传进骨头里的钝响。艾雷又击坠一机时，XGF-01A 的推进器也被碎片卷中，尾喷猛地一歪，整台机体在惯性里朝未知方向漂了出去。敌人的红点终于被甩在后方，可他的视野里只剩下越来越深的黑和系统一声接一声逼近极限的提示。
