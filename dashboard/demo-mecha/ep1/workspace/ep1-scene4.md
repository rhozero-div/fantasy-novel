# Scene 4: 残骸带漂流

推进器故障后的宇宙比战场更安静。驾驶舱气压在缓慢下降，警报声被系统自动压低之后，反而像一根细针持续扎在耳后。艾雷靠回座椅，才发现自己整条手臂都在发抖。XGF-01A 漂在残骸带深处，外部镜头里全是失速旋转的钢铁碎片，没有友军信号，也没有返航航线。

他试着重连通讯，只换来一片断断续续的噪音。正当他准备切断备用电源时，频道里忽然插进来一个陌生的男声，低而平静：“联合军少尉，你的推进器还能动吗？”

艾雷猛地坐直，手几乎本能地又压回武器键。外部镜头切换后，他看见一台完好的敌机悬停在不远处，机体表面没有继续瞄准的动作，反而像是在刻意保持距离。驾驶员很快报了名字。维克托·海因。艾雷记得这个声音，记得战场上那些几乎把他逼死的角度。

“你该补最后一枪。”艾雷盯着屏幕，说话时喉咙发紧。

通讯那头沉默了几秒，久到艾雷以为对方已经重新举枪。维克托才开口：“命令里没写必须回收残骸里的幸存者。”这句话听上去仍然像公事公办，可越是平，越能听出里面那一点被压住的迟疑。

艾雷没有放下戒备，却也没从这句话里听出杀意。维克托最后靠近了一些，把自己的机体姿态调整到能替他挡住碎片流的角度。“跟着我的信号漂。”他说，“出了这片残骸带，再决定要不要恨我。”

驾驶舱里沉默了很久。艾雷最终没有切断通讯。宇宙里没有任何真正的温度，可他知道，从这一刻开始，维克托已经站到了命令之外。
