# EP3

峡谷里的风不是从外面吹进来的，而像是从石壁旧伤里一寸寸渗出来。顾停灯在前面拨开藤蔓，没有回头，只说了一句：“再往里，别信你耳朵听见的东西。”林雾川没应声，只把手指按在腰间那枚潮音铜铃上。铃身隔着衣料发凉，像有一滴水一直停在骨缝里。

两人越往里走，石壁上的潮纹就越像被人用指甲一点点刮出来的。顾停灯忽然停步，侧耳听了片刻，低声说前面有人走过。林雾川还没来得及问，雾里就先传来一阵熟悉的笑声，像阿杳隔着很远的地方，在替谁把一句话咽了回去。

后来他们在裂开的石台前找到她。阿杳站在旧祭痕边，脚下全是积水照出来的碎光，神情却和平时一样轻，好像她只是偶然路过。她看着林雾川，把“这里什么都没有”说得像真的。可她回避他的目光时，手指却悄悄按在石台边缘，像在替某个不该醒来的东西遮住呼吸。

林雾川第一次没有追问到底。他只把铜铃收回袖中，记住了她那一下几乎看不见的迟疑。顾停灯在一旁沉默良久，最后把刀鞘重新扣回腰侧，只说他们该离开峡谷了。风从三人之间穿过去，带着潮湿石腥。林雾川知道，这一趟走出去以后，王城就不再只是一个远处的名字。
