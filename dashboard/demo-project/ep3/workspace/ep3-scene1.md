# Scene 1: 峡谷同行

峡谷里的风不是从外面吹进来的，而像是从石壁旧伤里一寸寸渗出来。顾停灯在前面拨开藤蔓，没有回头，只说了一句：“再往里，别信你耳朵听见的东西。”

林雾川没应声，只把手指按在腰间那枚潮音铜铃上。铃身隔着衣料发凉，像有一滴水一直停在骨缝里。两人脚下踩过碎石和烂叶，声音都被四周潮湿的石壁吃掉了，走得越深，越像是被峡谷整个含进了喉咙里。

顾停灯忽然停步，抬手示意他别动。前方一丛垂下来的藤蔓还在轻轻晃，显然有人刚走过去不久。他低声说：“不是山风。”这句话落下以后，两人之间的那点暂时结盟又更紧了一寸，却也更危险了一寸。

林雾川看着他侧脸，没有追问，只把另一只手从袖里伸出来，悄悄按住了短刃。两人继续往前时，谁也没有再说信任这种话，可他们都默认了眼下只能把后背交给彼此。
