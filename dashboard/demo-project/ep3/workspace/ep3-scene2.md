# Scene 2: 阿杳的隐瞒

再往前，雾里先传来一阵熟悉的笑声，轻得像有人隔着很远的地方替谁把一句话咽了回去。林雾川一抬头，就看见阿杳站在裂开的石台边，脚下是积水映出的碎光，肩头却半点水意也不沾。

“这里什么都没有。”她说得很快，像是怕他们多看一眼。顾停灯没接话，只把视线落在石台边缘那些刚被人摸过的潮痕上。林雾川却先看见了阿杳藏不住的小动作。她的手指压在石台裂缝旁，像在替某个不该醒来的东西遮住呼吸。

他本可以追问，可那句质问到了唇边，最后还是被压了回去。阿杳抬眼看他时，眼里第一次有了近乎恳求的闪避。林雾川于是只把铜铃收回袖中，低声说：“先出去再说。”

顾停灯沉默良久，最终把刀鞘重新扣回腰侧。他没有拆穿阿杳，也没有替她开脱，只说他们该离开峡谷了。风从三人之间穿过去，带着潮湿石腥。林雾川知道，这一趟走出去以后，王城就不再只是一个远处的名字。
