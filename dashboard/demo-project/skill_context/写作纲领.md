# 写作纲领

> 项目级，题材统一，写作时按此执行。
> 题材：修仙 / 山野异志 / 王朝秘案混合

---

## 题材定位

| 项目 | 内容 |
|------|------|
| 题材 | 山野少年卷入星图旧案，沿地域推进世界真相 |
| **分类** | 修仙 |
| **命名风格** | 中式古典 |
| 战斗语言 | 近身压迫感 + 灵息流动，不报数值 |
| 文戏语言 | 克制、留白、关系靠动作与视线推进 |
| 实力来源 | 吐纳、山灵感应、观象术 |

---

## 文戏质感

### 对话规则

- 角色说话要带身份差异
- 情绪通过停顿、避答、动作传递，不靠旁白替代
- 主角面对权威时先收，再反弹

### 内心独白

- 只在价值判断和恐惧临界点出现
- 不解释已经能从动作看出的内容

---

## 武戏质感

### 战斗节奏

- 遭遇→试探→错判→逆转→代价
- 力量感通过地形、呼吸、身体负担体现

### 逆转规则

- 逆转必须来自已埋下的能力或认知，不凭空爆种

---

## 题材特定规则

- 地域必须有气候/地形/秩序感，不是空舞台
- 技能名允许诗性，但使用效果必须清楚
- 每个 EP 至少推进一个“人”与一个“世界规则”的认知

---

## 全局约束

> 用户明确要求不能写的内容。所有写作阶段必须遵守。

**禁区：**

- 不写后宫
- 不写纯数值升级流
- 不让主角无代价开挂
