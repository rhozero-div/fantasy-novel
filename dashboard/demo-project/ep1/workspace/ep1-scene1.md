# Scene 1: 雾岭村的夜路

山雾压在村口石阶上，像一层终年不肯散的旧梦。林雾川背着药篓下山时，肩胛已经被潮气浸透。他一路没说话，只在经过祠前那株老槐时抬头看了一眼。树皮裂缝里嵌着旧红线，像被人缠住又挣不开的血管。

村口卖药的老郑把摊布一卷，隔着雾冲他摆手：“这点东西，明天再来吧。今夜风向不对。”林雾川嗯了一声，把药篓往肩上提了提，眼神却已经飘向后山那边。雾岭村的规矩他不是不知道，可规矩不能换米，也不能替他娘把咳嗽压下去。

他回到岔路口时，夜色已经沉得看不出山脊。风从坡下绕上来，带着草药晒过后的苦味，也带着一点说不清的冷，像有人隔着雾在看他。林雾川提起灯，灯芯只亮一豆，照不远，只够照见脚边碎石和枯草。

村里的人都说夜雾重的时候别回头，也别往禁路上看，可那条路偏偏总在视线边缘晃一下，像水里的一尾暗鱼。灯火在他掌心里摇了一下，他听见很轻的一声铃响，不像风碰出来的，更像有什么东西在雾后轻轻应了他一下。

林雾川站了很久，最后还是转过身，踩上那条平时没人走的碎石小路。药篓里的草药互相摩擦，发出干涩轻响，像在提醒他还可以回头。可他没有停。
