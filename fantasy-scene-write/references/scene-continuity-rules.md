# Scene 衔接规则

## 核心原则

衔接时，**弧光状态比具体动作更重要**。

好的衔接锚点：「张无忌已经用九阳神功逼退了崆峒派，空性正准备出绝招」
坏的衔接锚点：「张无忌使出了第七招龙爪手，空性侧身躲避」

## 衔接检查

每次承接前序输出时：

1. **读**：读完前序输出，总结弧光状态
2. **判**：这个弧光状态是否支持本 Scene 的弧光任务？
3. **接**：本 Scene 从哪个弧光节点开始？

## 弧光锚点格式

```
【前序弧光状态】
角色当前状态：

【本 Scene 任务】
从小位移 → 完成

【承接点】
（弧光节点描述）
```

## 常见衔接失败

| 失败类型 | 原因 | 修复方式 |
|---------|------|---------|
| 重复 | 前序已写过某个动作，本 Scene 又写一遍 | 锚点只给状态，不给动作序列 |
| 跳步 | 跳过了前序的弧光节点 | 检查衔接逻辑，确认物理顺序 |
| 弧光回退 | 写完 A→B，承接时回到 A | 重读前序，确认弧光方向 |
