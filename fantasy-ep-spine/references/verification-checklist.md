# EP Spine 验证清单

完成 Spine 后，逐条检查：

## 弧线完整性

- [ ] 主角弧线：入口状态明确（来自上一 EP 锚点）
- [ ] 主角弧线：核心位移明确（A → B）
- [ ] 主角弧线：终点状态明确
- [ ] 配角弧线：与主线弧光的关系标注清楚（从属/并行）
- [ ] 弧线类型已标注（战力/心境/关系/认知）

## Scene 结构

- [ ] 每个 Scene 有完整叙事节奏（起→承→转→合）
- [ ] Scene 之间有叙事停顿点
- [ ] Scene 数量与事件流分段匹配
- [ ] 物理事件（战斗/追逐/危机）未跨 Scene 拆分

## Scene 衔接

- [ ] 每个 Scene 有明确的叙事任务
- [ ] 相邻 Scene 之间有清晰的接续点
- [ ] 每个 Scene 能回答：「下一个 Scene 从哪里接？」

## 叙事逻辑

- [ ] 场景进入有契机
- [ ] 场景离开有条件
- [ ] Scene/Scene 之间的衔接逻辑清楚

## 锚点传递

- [ ] 本 EP 出口状态已标注（供下一 EP 使用）
