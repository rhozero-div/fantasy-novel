# 散文质感标准

> 跨题材通用。对所有幻想小说类别（日式异世界 / 修仙 / 武侠）均适用。

---

## 一、叙事声音

### 1. 写场景，不写分析

让读者看见人物在做什么、感受到什么。禁止作者跳出来解释人物心理或局势。

```
❌ 三成意识到自己已经走投无路了，他深知这是丰臣家最后的希望。
✅ 三成把信纸折好，火漆封口。窗外马蹄声渐远。
```

### 2. 用内在视角

从特定人物的感官出发。禁止上帝视角或「从某人的角度来看」这类提示词。

```
❌ 从艾伦的角度来看，这个决定太过冒险。
✅ 艾伦盯着地图上那条河。太冒险了。
```

### 3. 用对话和动作推进

不靠旁白解读。人物说了什么、做了什么，读者自己会判断。

```
❌ 他虽然嘴上答应，内心并不认同这个计划。
✅ 「明白了。」他把剑搁在桌上，没有松手。
```

### 4. 情感内敛

克制抒情，用具体行为暗示情绪。禁止直接说「他感到很难过/很愤怒/很绝望」。

```
❌ 他感到无比悲痛，眼泪止不住地流下来。
✅ 他转过身去。过了很久，才说：「走吧。」
```

### 5. 世界观信息编织在叙事中

世界观、背景知识通过对话和细节自然披露，不单独成段解释。

```
❌ 这个世界的修炼体系分为九个境界，从炼气期到渡劫期……
✅ 「炼气九层就想闯迷踪林？」掌柜头也不抬，「怕是连外围雾瘴都过不去。」
```

## 二、通用负面清单（跨题材适用）

写作时严禁出现以下 8 类句式。QC 阶段以正则扫描兜底。

### 1. 对空否定句式

禁止「不是 X 而是 Y」及任何变体（对着空气否定一个未提出过的观点）。

```
❌ 他瞄准的不是什么人，而是宇喜多阵前的一面马印。
✅ 他瞄准宇喜多阵前竖起的一面马印。

❌ 他并不是害怕战斗，而是担心连累同伴。
✅ 他看了一眼身后的同伴。「换个地方打。」
```

### 2. 直接称呼读者

禁止「你」「读到这里你也许会……」等。

```
❌ 如果你站在加藤清正的立场上看待这件事，你会觉得……
✅ 在加藤清正看来，这条命令并不意外。

❌ 你可能会觉得主角太过冲动。
✅ 事后回想，这个决定确实草率。
```

### 3. 作者旁白与总结

禁止作者跳出叙事直接下判断。

```
❌ 这是他的悲剧所在——忠诚既是他的优点，也是他的死因。
✅ 他的忠诚没有换来信长一句好话。他不再期待了。

❌ 这个选择改变了他的一生。
✅ 他选了左边那条路。他后来再也没见过村子。
```

### 4. 修辞设问引出下文

禁止以设问句作为段落的开头或过渡。

```
❌ 做成这件事需要什么？首先是勇气，其次是运气。
✅ 做成这件事需要勇气。需要一点运气。

❌ 为什么恨到这个程度？那要从三年前说起了。
✅ 这恨意始于三年前的那场大火。
```

### 5. 假设性想象框架

禁止声明「我们在想象」。直接写场景。

```
❌ 我们可以想象这样一个场景：大雨滂沱中，他独自站在城头。
✅ 大雨滂沱。他独自站在城头。

❌ 如果有人从天上俯瞰，会看到这幅画面——
✅ 从城楼上看下去，长街像一条被踩碎的蛇。
```

### 6. 空洞升华

禁止「意味着一切」「从未有人如此」「这才是真正的 X」等空泛定论句。收尾用具体意象，不升华。

```
❌ 这一刻，他意识到了友谊的真正意义。
✅ 他把那枚磨损的铜币放进怀里。「够了。」

❌ 这便是忍者最终的宿命。
✅ 月色下，屋顶只剩下一把断刃。
```

### 7. 设定标签化解释

禁止用作者口吻跳出来给设定下定义。

```
❌ 用当时的语言来说，这叫做「武士的矜持」。
✅ 「武士的矜持。」家老低声说，「他丢不起这个人。」

❌ 这个宗门的核心功法名为「九天玄火诀」，是一种火属性的高阶功法。
✅ 殿内热浪逼人。师兄说过，九天玄火诀练到第七层，连玄铁都能熔。
```

### 8. 现代对比打断

禁止拉现代概念做对比来衬托古代/异世界的落后。

```
❌ 在任何一个现代国家的军队里，都有一套制度来约束。但秀吉的体制里没有。
✅ 秀吉的体制没有约束。这就是问题本身。

❌ 在那个没有互联网的时代，消息传播全靠口口相传。
✅ 消息从城东传到城西，用了三天。
```

## 三、幻想题材质感补充原则

### 1. 世界观信息披露

- 魔法/技能/境界信息通过对话和实战披露，不单独成段说明
- 禁止在战斗中途插入世界观说明

### 2. 战斗描写

- 战斗节奏遵循 `combat-mode.md` 已定结构
- 技能/招式效果通过视觉、听觉、触觉描写传递，不堆砌效果词
- 力量差距通过对比和选择传递，不靠数值

### 3. 对话节制

- 每段对白至少有第二层意思（潜台词）
- 战斗中对白量参考 `writing-guidelines.md` 第五章

### 4. POV 纪律

- 默认视角：在场角色 POV 全开
- 显式写出「XXX 不开 POV」时，写作中不得切换至该角色内心

---

## 四、精修检查流程

完成写作或 QC 检测时，逐条过以下检查：

1. 扫描对空否定句式
2. 扫描读者称呼
3. 扫描作者旁白总结
4. 扫描修辞设问
5. 扫描假设性框架
6. 扫描空洞升华句
7. 扫描标签化解释
8. 扫描现代对比打断
