# EP{N+1} 输入

> EP{N} 已经完成了。下一章，你想让故事往哪里走？

## 🎤 语音输入推荐

同样推荐用语音。把你对下一章的直觉、冲动、不满、期待全部说出来，不需要先想清楚结构。

有时候你对上一章那种“还想更多”的感觉，就是下一章最好的起点。

---

## 必须告诉我的

### 从 EP{N} 结束的地方，主角要走向哪里？

比如：离开新手村去更广阔的世界 / 遭遇重大背叛从此黑化 / 修炼遇到瓶颈必须下山历练……

> 不需要写具体场景，告诉我一个方向就行。

### 涉及地域（必填）

请列出本 EP 新增或涉及的地域：

- 新增地域：地域名称 + 简要说明
- 已有地域：如果故事回到之前 EP 的某个地方，不需要重新说明（系统会从地域锚点读取）

> 地域锚点在写作阶段自动生成或更新。

### 有什么一定不能写的吗？

> 这是你的故事，你有权利划定边界。

---

## 如果你愿意多说一点

- **关键事件**：这一章有什么一定要写的场景或冲突？
- **角色变化**：有没有想让某个角色在这一章发生转变？
- **新增对象**：有没有新技能、新宝物、新势力或新地域想在这一章引入？
- **其他想法**：任何你想让故事包含的东西。
