# EP1 输入

> 你内心那个故事是什么样的？不需要写得很完整，只要把你想到的、感受到的告诉我。

## 🎤 语音输入推荐

如果你觉得打字太慢或不想整理思绪，**直接用语音录下你想到的一切**。不管多零碎、多跳跃都没关系，想到什么就说什么。

我们的工具会替你梳理这些碎片，把它们变成可执行的章节设计。

**你只管说出来，剩下的交给我。**

---

## 必须告诉我的

### 你想讲一个什么故事？

比如：少年在宗门中一步步成长 / 异世界冒险者从底层爬升 / 武侠人物卷入江湖恩怨 / ……

> 不需要写完整大纲，几个关键词、一句话都可以。我会帮你整理成可继续执行的输入。

### 故事分类（必选）

请选择最适合你故事的分类：

- **日式异世界** — 勇者/冒险/魔法/技能名/社交地位变化
- **修仙** — 宗门/境界/顿悟/法宝/五行
- **武侠** — 江湖/招式名/内功/克制情感
- **机甲** — 巨型机器人/宇宙战争/军事科幻/政治冲突

> 这个选择会影响写作时的语言风格、战斗描写方式和整体氛围，必须选。

### 命名风格（必选）

请选择 NPC 命名风格：

- **中式古典** — 如 林黛玉、贾宝玉、令狐冲
- **中式现代** — 如 李明、王芳、张伟
- **日式** — 如 佐藤、田中、鈴木
- **西式** — 如 艾琳、莱恩、诺拉

> 主角名字会作为风格锚点，NPC 使用同类风格。

### 涉及地域（必填）

请列出本 EP 故事涉及的地域：

- 地域名称（如「边境小镇」「王城」）
- 简要说明（如「商路交汇点」「高海拔军镇」）

> 地域锚点在写作阶段自动生成或更新。

### 你的主角是谁？

名字、身份、或者只是一个模糊的印象，比如“一个从小村庄出来的孤儿”或者“一个被退婚的少年”。

> 只要让我知道你在乎的是什么样的角色。

### 有什么一定不能写的吗？

比如：不想写女主、不能接受角色死亡、某些情节让我不舒服……

> 这是你的故事，你有权利划定边界。

---

## 如果你愿意多说一点

以下都可以自由补充，越多越能帮你做出你想要的东西。不填也没关系。

- **其他角色**：有没有什么配角已经在你脑海里了？他们的名字、身份、和主角的关系。
- **设定**：力量体系、宗门/门派、宝物、技能……有没有想好？
- **EP1 走向**：你想让主角先经历什么？
- **任何其他想法**：世界观的碎片、想写的经典桥段、参考的作品……
