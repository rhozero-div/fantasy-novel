# EP 锚点

> 全局汇总表。EP{N} 完成后写入本表「出口状态」列，供 EP{N+1} 读取作为入点。EP{N} 进行中时本表对应行留空。

## 填写时机

| 填写时间 | EP{N} 出口状态来源 |
|---------|-----------------|
| EP1 完成前 | EP1 行留空，EP1 入点来自 `user_input.md` |
| EP1 完成后 | 写入 EP1 主角/配角出口状态 |
| EP2 完成前 | EP2 入点读取 EP1 行 |
| EP2 完成后 | 写入 EP2 主角/配角出口状态 |
| ... | ... |

## EP 出口状态

| EP | 主角 | 主要配角 |
|----|------|---------|
| 1 | （EP1 完成后填写） | （EP1 完成后填写） |
| 2 | （EP2 完成后填写） | （EP2 完成后填写） |
| 3 | （EP3 完成后填写） | （EP3 完成后填写） |
