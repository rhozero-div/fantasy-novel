# 战斗 Mode

---

## 框架：按力量对比分三层

| 类别 | 描述 | Mode |
|------|------|------|
| **俯视型** | 我强于对方 | 碾压型、围猎型 |
| **平视型** | 势均力敌 | 对抗型 |
| **仰视型** | 我弱于对方 | 反杀型、奔逃型、防守型、暗杀型 |

---

## 俯视型（我强于对方）

### 碾压型

**底色：** 绝对力量差距，结果无悬念。

**判定：** 一方实力远超另一方，过程是「确认差距」而不是「争取胜利」。

**张力来源：** 角色如何面对无法战胜的对手？认输？逃跑？拖延？还是用其他方式应对？

**对白特征：** 弱势方抵抗台词、强势方宣告台词。

---

### 围猎型

**底色：** 多人围攻一人或一方围剿另一方。

**判定：** 数量优势明显，但被围者有其他优势（地形、技能、意志）。

**张力来源：** 围猎方的配合、被围者的绝地反击。

**对白特征：** 围攻者轮番施压、被围者的孤注一掷。

---

## 平视型（势均力敌）

### 对抗型

**底色：** 双方势均力敌，有来有回。

**判定：** 双方都有赢面，过程充满不确定性。

**张力来源：** 每一次攻守转换都有胜负悬念。

**对白特征：** 攻守双方的语言博弈，谁先开口谁就暴露意图。

---

## 仰视型（我弱于对方）

### 反杀型

**底色：** 弱势方逆转。

**判定：** 初期绝对劣势，通过某个触发点逆转局面。

**触发点设计：** 为什么能反杀？顿悟？触发隐藏条件？对手失误？必须有弧光理由。

**张力来源：** 反杀的条件和时机。

**对白特征：** 从绝望到反击的台词变化。

---

### 奔逃型

**底色：** 主动撤退或被动被追。有目的的边打边撤，或被追着打。

**判定：**
- 有主动选择 → 撤退（断后/掩护/保人）
- 无主动选择 → 追杀（被追着打）

**张力来源：** 能否成功脱身？断后者能否撑住？

**对白特征：** 撤退指令/求援信号、追杀方的封锁宣言。

---

### 防守型

**底色：** 死守据点/人物，坚持到转折。

**判定：** 有必须守护的目标（城池/人物/秘密），守方处于被动承受状态。

**张力来源：** 能不能守住？援军何时到来？

**对白特征：** 誓死守护的宣言、绝望中的坚持。

---

### 暗杀型

**底色：** 目标不是打败，是杀死特定人物。

**判定：** 战斗的终点是对方死亡，不是战斗力归零。

**张力来源：** 暗杀能否成功？目标如何防范？

**对白特征：** 暗杀者的宣言、目标的警觉或无觉。

---

## 判定流程

```
这场战斗的力量对比是？

1. 我强于对方？
   → 是 → 有人数优势吗？
        → 是 → 围猎型
        → 否 → 碾压型
   → 否 ↓

2. 势均力敌？
   → 是 → 对抗型
   → 否 ↓

3. 我弱于对方？
   → 是 → 战斗的目的是杀死对方？
        → 是 → 暗杀型
        → 否 → 有必须守护的目标？
             → 是 → 防守型
             → 否 → 能反杀？
                  → 是 → 反杀型
                  → 否 → 奔逃型
```

---

## 与 Scene Arc 的关系

战斗 Mode 回答「力量对比是什么」。

战斗戏的 Scene Arc（位移逻辑）可以是：

- 碾压型 + 外压型（被碾压，状态被压垮）
- 对抗型 + 博弈型（势均力敌的对抗）
- 反杀型 + 内爆型（逆转来自内心触发）
- 围猎型 + 外压型（被围猎，被动承受）
- 奔逃型 + 积累型（多次逃脱尝试）
- 防守型 + 积累型（坚持到临界点）
- 暗杀型 + 揭示型（身份被揭露后的刺杀）

## 战斗 Mode × 题材差异

| Mode | 日式异世界 | 修仙 | 武侠 |
|------|-----------|------|------|
| **碾压型**（俯视） | 等级碾压，属性克制，一招定胜负。「你连让我认真对付的资格都没有。」 | 境界压制，元婴杀金丹如捏蝼蚁。「萤火之光，也敢与日月争辉。」 | 内功差距，门派底蕴悬殊。「你这辈子都破不了我的防。」 |
| **围猎型**（俯视） | BOSS召唤小怪，迷宫轮番围攻。「别让他跑了！」 | 宗门围攻，法阵困杀，群狼战术。「今日必取你项上人头。」 | 门派群殴，帮主车轮战。「少林十八罗汉阵，起！」 |
| **对抗型**（平视） | 技能互换，属性相克反复逆转。「有意思……再来！」 | 法宝对轰，法则碰撞，符箓互换。「你的道，还不够硬。」 | 招式拆解，兵器试探，有来有回。「好快的剑，好稳的手。」 |
| **反杀型**（仰视） | 濒死属性觉醒，等级压制逆转。「这种程度……还不够！」 | 心魔突破，顿悟天道，境界顿升。「道心不灭，何惧此劫。」 | 以命换招，对方露出破绽。「这一剑，我等了十年。」 |
| **奔逃型**（仰视） | BOSS追击，技能冷却逃命。「我来断后，快跑！」 | 大能追杀，血遁大法，天道锁定。「他日必将今日之辱，百倍奉还。」 | 高手追杀，断后赴死。「你们先走，我来挡！」 |
| **防守型**（仰视） | 迷宫大门死守，召唤塔防。「守住三十秒，援军就到！」 | 护山大阵，以阵耗敌，丹药续命。「只要护山大阵不破，他就进不来。」 | 死守山门，师兄断臂。「我少林寺纵剩一兵一卒，也绝不投降。」 |
| **暗杀型**（仰视） | 刺客工会，技能前摇，一击脱离。「你甚至不知道我是谁。」 | 夺舍暗杀，元神灭杀，暗杀道修士。「死在自己的贪念里，倒也合适。」 | 唐门暗器，毒杀，一击必退。「我的刀，从来只出一次。」 |

### 题材核心差异速查

| 差异维度 | 日式异世界 | 修仙 | 武侠 |
|---------|-----------|------|------|
| **实力来源** | 等级/技能树/装备 | 境界/灵根/功法 | 内功/招式/兵器 |
| **战斗语言** | 属性面板、被动技能、觉醒 | 天道法则、顿悟、丹药品级 | 门派传承、招式名称、内力深浅 |
| **碾压感的来源** | 等级差太大，对方无法破防 | 境界压制，元婴期看筑基期如蝼蚁 | 内功差距悬殊，一掌震断对方兵器 |
| **逆转感的来源** | 隐藏属性激活，等级压制逆转 | 心魔突破或大能附身 | 悟出新招，以命换破绽 |
| **追杀感的来源** | BOSS技能锁定，地图机制 | 血遁/天劫锁定，无法逃脱 | 轻功差距，追方总能堵住逃方 |
| **暗杀感的来源** | 隐身技能，暗杀者被动，一击脱离 | 元神攻击，夺舍，隔空灭杀 | 暗器/毒杀，身份成谜 |
