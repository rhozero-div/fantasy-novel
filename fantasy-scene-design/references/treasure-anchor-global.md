# 全局宝物锚点模板

> 项目级，所有 EP 共享。实时更新，EP 完成后不合并。

## 格式

```markdown
# 宝物锚点

| 宝物 | 持有者 | 定性阶段 | 核心能力 | 首次出现EP |
|------|--------|---------|---------|-----------|
| 霜纹长枪 | 主角 | 认主 | 寒锋裂阵 | EP2 |
| 幽冥残卷 | 配角A | 炼化中 | 摄魂扰心 | EP3 |
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| 宝物 | 宝物名称，全局唯一 |
| 持有者 | 当前持有者，同一时间唯一 |
| 定性阶段 | 未认主/认主/炼化中/完全掌控 |
| 核心能力 | 写作时查，无具体数值 |
| 首次出现EP | 便于追溯来源 |

---

## 变更规则

- **宝物获得**：新增一行，注明首次出现EP
- **认主程度变化**：更新定性阶段，注明变更EP
- **易主**：更新持有者，注明变更EP和弧光原因
- **宝物损毁/消失**：移除记录，原记录保留（划掉）

**易主时更新：**
1. `skill_context/宝物锚点.md`（全局）
2. `skill_context/人物锚点.md`（相关角色战力弧变化）

---

## 与人物锚点的关系

宝物易主/升级时 → 同时更新：
1. `skill_context/宝物锚点.md`
2. `skill_context/人物锚点.md`（战力弧状态）
