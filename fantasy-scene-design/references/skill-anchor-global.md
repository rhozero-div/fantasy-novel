# 全局技能锚点模板

> 项目级，所有 EP 共享。实时更新，EP 完成后不合并。

## 格式

```markdown
# 技能锚点

| 技能 | 持有者 | 定性阶段 | 标志性效果 | 首次出现EP |
|------|--------|---------|-----------|-----------|
| 炎狱术 | 主角 | 大成 | 炽焰裂空，剑光成环 | EP1 |
| 幽影步 | 配角A | 小成 | 残影分身，难辨真身 | EP3 |
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| 技能 | 技能名称，全局唯一 |
| 持有者 | 修炼此技能的角色，可多对多 |
| 定性阶段 | 入门/初成/小成/大成/圆满，按项目自定词 |
| 标志性效果 | 写作时查，无具体数值 |
| 首次出现EP | 便于追溯来源 |

---

## 变更规则

- **技能获得**：新增一行，注明首次出现EP
- **技能升级**：更新定性阶段，注明变更EP
- **技能遗失/替换**：持有者变更，原记录保留（划掉）或移除
- **EP内变更时更新**：同时更新 `skill_context/技能锚点.md`

---

## 与人物锚点的关系

战力弧变化时（如技能升级）→ 同时更新：
1. `skill_context/技能锚点.md`
2. `skill_context/人物锚点.md`（战力弧状态）
